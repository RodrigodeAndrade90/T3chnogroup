# Projeto-InFatec

```bash
git clone #
cd Projeto-InFatec
```
Para Windows:
```bash
python3 -m venv env
.\env\Scripts\activate
```

Para Linux:
```bash
python3 -m venv env
source env/bin/activate
```

Instalar dependencias:
```bash
pip3 install -r requirements.txt
```

Executar aplicação:
```bash
python3 wsgi.py
